Quy trình thực hiện 
1. Xác định bài toán
2. Tìm hiểu về luật kết hợp

- Các độ đo (Support, Confidence, Lift)
- Ứng dụng
- Các khai thác
3. Thuật toán Apriori
- Mục đích
- Trình tự thực hiện 
4. Tìm kiếm bộ dữ liệu
  - Giải thích bộ dữ liệu
  - Tiền xử lý
  - Trực quan dữ liệu
 5. Xây dựng mô hình bằng Python
    - Lựa chọn cột
    - Tính các chỉ số
    - Xây dựng các luật
    - Vẽ mô hình mạng noron
    - Giải thích kết quả
